○説明

AtCoder で AHC とそれ以外のコンテストの問題文を同時に開いている場合に、後者の提出欄に警告文を表示します。
開催中の AHC の解答コードの、過去問などへの誤提出による規約違反を防ぐことを目的としています。

厳密な仕様
・URL に「atcoder」と「ahc」の両方の文字列を含むサイトをあるウィンドウのあるタブで開いている場合に、
・URL に「atcoder」を含むが「ahc」を含まないサイトにおいて、
・通常の提出欄にあるものと同じ (html の) クラスが存在する場合に、
・そこに「【警告】これは AHC ではありません！」という文字列を赤色の太字で表示します。

注意
・規約違反の可能性がある誤提出が起こり得る場合であっても、警告文字列が表示されないことがあります。例えば「URL に『ahc』を含まない開催中のヒューリスティックコンテスト」と「ABC 過去問」のみを同時に開いた場合、後者の提出欄に警告文字列は表示されません。
・規約違反の可能性がある誤提出が起こり得ない場合であっても、警告文字列が表示されることがあります。例えば「AHC 過去問」と「ABC 過去問」のみを同時に開いた場合、後者の提出欄に警告文字列が表示されます。
・今後の AtCoder の仕様変更により、動作しなくなる可能性や、想定しない動作をするようになる可能性があります。
・本拡張機能の使用により、誤提出などによる規約違反やコンテストのスコア・順位の低下などの不利益が発生した際に、本拡張機能の作成者は責任を負いません。

○単一用途の説明

競技プログラミングのサイト AtCoder において、コンテストに参加する場合、コンテスト問題を解くためのソースコード (以下、解法コード) を、AtCoder の Web ページを通してコンテスト期間中に何回か提出することになる。コンテスト期間中は、原則として解法コードを他者から見られる場所に公開してはならないと、規約で定められている。
一方、AtCoder では過去のコンテスト問題を解くことができ、その解法コードは提出後即座に Web 上に公開される。そのため、開催期間中のコンテスト問題のタブと過去のコンテスト問題のタブを Web ブラウザで同時に開いている場合、前者へ提出すべき解法コードを誤って後者に提出してしまい、意図せず規約に違反してしまうおそれがある。
本拡張機能では、AtCoder の複数タブを同時に開いている一部の状況において、過去問題の可能性が高いタブの提出ボタン付近に警告文を表示することで、解法コードの意図しない公開行為を防止する。

○tabs が必要な理由

特定条件を満たす AtCoder の複数タブを同時に開いているかを判別することで、解法コードの意図しない公開行為が起こりそうな状況であるかを判断するため。

○scripting が必要な理由

過去問題の可能性が高いページに、警告文を表示するスクリプトを挿入するため。

○ホスト権限 が必要な理由

AtCoder の一部ページに、警告文を表示するスクリプトを挿入するため。
