# AtCoder Parallel Alert

<a href="https://atcoder.jp/">AtCoder</a> で AHC とそれ以外のコンテストの問題文を同時に開いている場合に、後者の提出欄に警告文を表示する Chrome 拡張機能です。  
開催中の AHC の解答コードの、過去問などへの誤提出による規約違反を防ぐことを目的としています。
